Game Instructions

  1. Start the game by clicking any key from keyboard

  2. By default, the game uses Random Pattern

  3. Make sure to add custom patten by clicking 'Add Custom Pattern' button to add your custom pattern

  4. By default, the custom patten will be removed every time when you loose or win a game

  5. If you win or lose the current game, then next game will start as 'Random Pattern' unless you add custom pattern before next game starts

  6. Please contact us, if you run into any bugs during game or website itself
